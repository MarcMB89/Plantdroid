# Plantdroid

## React-Native Plant-care App

To use the App on your emulator, go to VSCODE and open the terminal:
- git clone https://github.com/MarcMB89/Plantdroid
- cd .\Plantdroid\
- cd cd .\frontend\
- npm install
- npm run android
Open a different terminal:
- cd .\backend\ 
- npm install
- npm start
